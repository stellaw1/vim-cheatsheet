# Vim Cheatsheet

One commange a day, keeps the confusion away. 

## File
| Command       | Usage             |
| ------------- | ----------------- |
| :q            | Quit              |
| :q!           | Quit without save |
| :wq           | Save & Quit       |
| :set number   | Show line numbers |
| :set nonumber | Hide line numbers |

## View
| Command       | Usage             |
| ------------- | ----------------- |
| Ctrl + f      | Page down         |
| Ctrl + b      | Page up           |


## Edit
| Command       | Usage             |
| ------------- | ----------------- |
| i             | Insert            |
| a             | Append            |
| dd            | Delete line       |
| yy            | Copy line         |
| p             | Paste             |
